
IGOOR est doté de trois types de mémoire :

- [[2 - Mémoire extraite depuis vos documents]] (manuelle, statique)
- [[3 - Mémoire de long terme]] (automatique, incrémentale)
- [[4 - Mémoire de court terme]] (automatique, dynamique)

Ces mémoires sont utilisées de façon différente pour personnaliser les prédictions du logiciel :

- la première est fournie par vous
- les deux autres sont automatiquement ajoutées par l'IA en fonction de vos conversations à travers le logiciel IGOOR
