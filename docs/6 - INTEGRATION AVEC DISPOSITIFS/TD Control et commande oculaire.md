Si vous utilisez IGOOR avec un logiciel tel que TD Control par Tobii, vous pouvez mettre en place une petite astuce pour éviter beaucoup de clics supplémentaires.

**IMPORTANT: L'association IGOOR n'est PAS associée avec Tobii / Dynavox.**

Dans TD Control, en effet, par défaut à chaque fois que vous voulez intéragir avec un bouton, TD Control vous demande quel type d'action vous souhaitez mener (ex. clic gauche, clic droit, glisser déposer etc.)

Cependant, à l'intérieur du logiciel IGOOR tout se fait avec le clic gauche. 

Voici donc quelques conseils :

Pour activer/ désactiver le clic continu dans TD Control, c'est le bouton qui est à droite du bouton pause (dans le menu hors champ) : il s'agit d'un + entouré de 2 flèches circulaires.

De base, il permet de répéter le clic gauche, le cliquer glisser et maintenu glisser.

S'il ne se passe rien, la fonction est peut-être désactivée. Dans le menu, réglages, aller dans Personnaliser, Mode Continu, choisir Cliquer lors de la fixation.

Ici, on peut aussi ajouter un autre bouton.

Pour info, il y a aussi **un mode Répété qui permet :**

- **d'activer/désactiver le clic gauche uniquement**
    
- **survoler sans cliquer** 
    
- **masquer la trace du regard**
