## Paramètres obligatoires

La première fois que vous ouvrez IGOOR, la page des paramétrages s'ouvre automatiquement :

![Onboarding](../assets/igoor_onboarding.png)

Vous devez sauvegarder tous les champs obligatoires.

### L'état de santé 

L'état de santé est un champ crucial parce qu'il contraint systématiquement les prédictions fournies par l'I.A. sur la base des capacités physiques de l'utilisateur. Si ces capacités évoluent sensiblement, il est bien de le mettre à jour meme si la mémoire de IGOOR pourrait contenir déjà ces informations.
